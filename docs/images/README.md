# 图片资源文件夹

这个文件夹用于存储文档中使用的图片资源。

## 使用说明

1. 使用 VS Code 的 okmd 插件粘贴图片时，图片会自动上传到云端
2. 生成的链接可以直接在 Markdown 文件中使用
3. 本地图片请放在这个文件夹中，并使用相对路径引用

## 图片命名规范

- 项目截图：`project_[项目名]_[功能]_[日期].png`
- 界面截图：`ui_[页面名]_[日期].png`
- 技术图表：`tech_[技术名]_[日期].png`

例如：
- `project_unity_gameplay_2025-07-08.png`
- `ui_dashboard_2025-07-08.png`
- `tech_ue5_blueprint_2025-07-08.png`
