# 项目介绍

## 社团项目概览

ArtTech 社团致力于将艺术与技术相结合，开发具有创新性和实用性的项目。我们的项目涵盖游戏开发、数字艺术、技术工具等多个领域。

## 当前项目状态

### 🎮 游戏开发项目

#### 正在开发中的项目

**《幻境冒险》** - UE5 动作冒险游戏
- **开发阶段**: 原型开发
- **技术栈**: UE5, C++, 蓝图
- **团队规模**: 6人
- **预计完成**: 2025年12月
- **项目描述**: 一款以魔法世界为背景的第三人称动作冒险游戏，玩家将扮演年轻的法师，在广阔的开放世界中探索、战斗和成长。

**《极速追击》** - UE5 赛车游戏
- **开发阶段**: 系统开发
- **技术栈**: UE5, 蓝图, Niagara
- **团队规模**: 4人
- **预计完成**: 2025年10月
- **项目描述**: 高速赛车竞技游戏，重点关注真实的物理驾驶感受和炫酷的视觉特效。

**《像素大冒险》** - Unity 2D 平台游戏
- **开发阶段**: 关卡设计
- **技术栈**: Unity 2D, C#
- **团队规模**: 5人
- **预计完成**: 2025年9月
- **项目描述**: 经典像素风格的横版动作冒险游戏，结合解谜和动作元素。

**《跑酷英雄》** - Unity 3D 跑酷游戏
- **开发阶段**: 原型测试
- **技术栈**: Unity 3D, C#
- **团队规模**: 4人
- **预计完成**: 2025年11月
- **项目描述**: 无限跑酷游戏，玩家需要在不断变化的城市环境中奔跑和跳跃。

#### 已完成项目

**《迷宫探索者》** - UE4 解谜游戏
- **完成时间**: 2024年12月
- **获得成就**: 校内游戏开发竞赛第二名
- **技术亮点**: 程序化迷宫生成、AI 行为系统

**《射击训练场》** - UE4 FPS 训练
- **完成时间**: 2024年8月
- **用途**: 教学演示项目
- **技术亮点**: 精确弹道计算、成绩统计系统

**《弹珠消除》** - Unity 2D 休闲游戏
- **完成时间**: 2024年11月
- **平台**: Android
- **技术亮点**: 物理引擎应用、关卡编辑器

**《塔防守护》** - Unity 2D 策略游戏
- **完成时间**: 2024年9月
- **获得成就**: 校内竞赛优胜奖
- **技术亮点**: AI 寻路算法、平衡性调优
